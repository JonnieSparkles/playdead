Album Specification

Each album should include the following files and structure:

1. album.json

The metadata file describing the album. Example:

{
    "band": "Grateful Dead",
    "title": "Live at Barton Hall",
    "date": "May 8, 1977",
    "source": "Audience recording",
    "info": "more_info.txt",
    "tracks": [
        {"number": 1, "title": "New Minglewood Blues"},
        {"number": 2, "title": "Loser"}
    ]
}

Note: The tracks field is optional. If omitted, the app will auto-generate track titles based on filenames.

2. Tracks Folder

Place all track files in a Tracks/ folder.

Naming Format: Use numbers to indicate track order followed by the track name in any style, e.g., 01 Track Name.mp3 or 1-Track Name.flac.

Supported Formats: .mp3, .flac, .wav

3. Album Cover

Include an image file named album_cover.png for the album's cover art.

Recommended size: 600x600 pixels (optional, but preferred).

4. Additional Information (Optional)

Add a file (e.g., more_info.txt) for album notes, recording details, or other information. Reference this in the info field of album.json.

Folder Structure Example

/Album
├── album.json
├── album_cover.png
├── more_info.txt
├── Tracks/
│   ├── 01 Track Name.mp3
│   ├── 2 - Another Track Name.flac
│   ├── 3.TrackName.wav

How to Submit

  1. Upload the Album Folder to Arweave: Use your preferred Arweave client (e.g., ArDrive) to upload the album folder and its contents.
  2. Create the Album Manifest (album_manifest): Generate a manifest for the album folder: 
    {
    "manifest": "arweave/paths",
    "version": "0.1.0",
    "index": {
        "path": "album.json"
    },
    "paths": {
        "album.json": { "id": "TXID_OF_ALBUM_JSON" },
        "album_cover.png": { "id": "TXID_OF_ALBUM_COVER" },
        "Tracks/01 Track Name.mp3": { "id": "TXID_OF_TRACK" },
        "Tracks/2 - Another Track Name.flac": { "id": "TXID_OF_TRACK" },
        "more_info.txt": { "id": "TXID_OF_INFO_FILE" }
    }
}
  3. Share the Manifest TXID: Provide the album_manifest TXID for inclusion in the catalog.
