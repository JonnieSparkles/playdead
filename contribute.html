Contribute to ar://playdead

About ar://playdead

ar://playdead is a community-driven platform that celebrates and preserves music forever. Albums uploaded here are saved on Arweave, a decentralized network that ensures permanent, uncensorable storage. Whether you're sharing audience recordings or public domain music, your contributions will live on as part of this timeless archive.

Contribution Guidelines

We accept audience recordings and music that is explicitly open-source or in the public domain. Please ensure that any music you upload respects these guidelines to honor the artists and their communities.

Album Specification

Each album should include the following files and structure:

1. album.json

The metadata file describing the album. Examples:

Audio Album:

{
    "version": "1.1.0",
    "type": "audio",
    "band": "Grateful Dead",
    "title": "Live at Barton Hall",
    "date": "May 8, 1977",
    "source": "Audience recording",
    "info": "more_info.txt",
    "tracks": [
        {"number": 1, "title": "New Minglewood Blues"},
        {"number": 2, "title": "Loser"}
    ]
}

Video Album:

{
    "version": "1.1.0",
    "type": "video",
    "band": "Grateful Dead",
    "title": "Live at The Centrum",
    "date": "April 9, 1988",
    "source": "Audience recording",
    "info": "more_info.txt",
    "reels": [
        {"number": 1, "title": "Complete Show", "duration": 8400}
    ]
}

2. Media Files

Place media files in the appropriate folder:

  • Audio: Place all track files in a Tracks/ folder.
  • Video: Place all video files in a Reels/ folder.

Naming Format: Use numbers to indicate track/reel order followed by the name in any style:

  • Audio: 01 - Track Name.mp3 or 1_Track Name.flac
  • Video: 01 - Show Part 1.mp4 or 1_Complete Show.webm

Supported Formats:

  • Audio: .mp3, .flac, .wav
  • Video: .mp4, .webm

Folder Structure Examples

/Audio Album                      /Video Album
├── album.json                    ├── album.json
├── album_cover.png               ├── album_cover.png
├── more_info.txt                 ├── more_info.txt
├── Tracks/                       ├── Reels/
│   ├── 01 Track Name.mp3         │   ├── 1 - Complete Show.mp4
│   ├── 2 - Another Track.flac    │   └── 2 - Encore.mp4
│   └── 3.Track Name.wav

How to Submit

  1. Upload the Album Folder to Arweave: Use your preferred Arweave client (e.g., ArDrive) to upload the album folder and its contents.
  2. Create the Album Manifest (album_manifest): Generate a manifest for the album folder: 
    {
    "manifest": "arweave/paths",
    "version": "0.1.0",
    "index": {
        "path": "album.json"
    },
    "paths": {
        "album.json": { "id": "TXID_OF_ALBUM_JSON" },
        "album_cover.png": { "id": "TXID_OF_ALBUM_COVER" },
        "Tracks/01 Track Name.mp3": { "id": "TXID_OF_TRACK" },
        "Tracks/2 - Another Track Name.flac": { "id": "TXID_OF_TRACK" },
        "more_info.txt": { "id": "TXID_OF_INFO_FILE" }
    }
}
  3. Share the Manifest TXID: Provide the album_manifest TXID for inclusion in the catalog.
